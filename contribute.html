Contribute to this document

These data were automatically extracted and parsed from a pdf, and many mistakes may remain. You are very welcome to contribute! Here is what you can do:

  • Fix a mistake in one of the tables: edit the corresponding csv file
  • Add a missing pdf: just add it in the pdf/ repository naming it refnumber.pdf (eg 84a.pdf or 26.pdf), and a link will automatically be generated
  • Add a missing link to ecocyc: add a new line to ecocyc.csv linking the name used in our tables with and an ecocyc ID, and a link will automatically be generated

After doing these any of these changes, you can just run make to update larossa.html. If you are satisfied with the outcome, send a patch / a pull request. Other contributions and suggestions of improvement are also very welcome!
